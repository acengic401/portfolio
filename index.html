CIS 150

This course provides an overview of contemporary information systems, and how they are used in organizations. It presents an introduction to the discipline of CIS and how information systems can be used to create competitive advantage. Topics include application/web development, including the system development life cycle, information security, business process management, and emerging technologies.

CIS 199

This course introduces object-oriented concepts such as the use of classes, methods, encapsulation, and inheritance. The course concentrates on using object-oriented programming to solve simple problems involving input and output. Below, I have attached screenshots of some of the C# applications that I created in this class. In the first assignment, I created a console application that would calculate the materials and labor costs for a carpet sales and installation company. This assignment required the use of variable and simple arithmetic. In the second assignment, I created a GUI application that would calculate income taxes based on different income brackets. This assignment required the use of decision-making logic. CIS 199 Link

CIS 200

This course was a continuation of CIS 199 and it delved deeper into object-oriented software development concepts such as polymorphism, inheritance, object interaction, and data structures. In the first program, I created a simple class hierarchy for package delivery services in order to calculate the cost of shipping individual items based on the shipping address and weight of the package. I used object oriented programming concepts such composition and inheritance in this application. In the second program, I generated reports that sorted parcels based on their destination zip code, shipping costs, and weight. I used LINQ in order to create these queries. CIS 200 Link
